Básicos de la programación en p5.js

Conceptos fundamentales para aprender a programar de forma visual y creativa.

1. ¿Qué es la programación?

La programación es el proceso de crear un conjunto de instrucciones que le dicen a una computadora como realizar algún tipo de tarea.

El código se ejecuta de arriba hacia abajo

(igual que la lectura en la mayoría de lenguas occidentales).

El orden sí importa.

  • Automatizar procesos y crear aplicaciones, juegos y arte.
  • Los lenguajes tienen reglas sintácticas propias.
  • Usaremos JavaScript con p5.js (visual y creativo).

2. Variables

Una variable es un contenedor que almacena información que puede cambiar mientras el programa se ejecuta

Para crear una variable usamos let seguido del nombre.

let x = 50;  // Guardamos un número en la variable x

function draw() {
  x = x + 1;  // Actualizamos el valor para mover un objeto
}

Ejemplo básico

3. Condicionales

Nos permiten tomar decisiones dentro del programa. 

let x = 0;

function draw() {
  x = x + 1;

  if (x > 200) {
    fill(255, 0, 0);
  } else {
    fill(0, 0, 255);
  }

  circle(x, 200, 50);
}

if…else if

if (x < 100) {
  // primera condición
} else if (x < 200) {
  // segunda condición
} else {
  // si ninguna se cumple
}

5. Bucles

Un bucle permite repetir acciones muchas veces sin escribirlas manualmente. 

function draw() {
  background(220);

  for (let x = 0; x < width; x = x + 20) {
    line(x, 0, x, height);
  }
}

6. Ejercicio: crear un sketch con ChatGPT

Vamos a crear un sketch con ChatGPT y vamos a pedirle que nos explique cada proceso.

Objetivo del ejercicio

  • Diseñar un sketch en P5.js que contenga los coneptos aprendidos(variables, condicionales, operadores, bucles y funciones)
  • Trabajar con ChatGPT como herramienta de aprendizaje y desarrollo que permite mejorar nuestro código en varias iteraciones.
  • Comprender completamente el código resultante y poder explicarlo con tus propias palabras.

Instrucciones

Requisitos mínimos

  • Al menos 3 versiones diferentes del código (3 actualizaciones respecto al original).
  • Uso de:
    • Al menos una variable propia (creada con let).
    • Al menos un condicional (if, else if, else).
    • Al menos un operador lógico (&&, || o !).
    • Al menos un bucle (for, while o repetición visible).
    • Al menos una función creada por ti (por ejemplo function dibujarForma() ).
  • Sketches funcionando en el editor de p5.js. Hacerse cuenta en P5.js (a poder ser con el correo de educamadrid). Enviar "Collection" que contenga los Sketches correspondientes. Esta opción puede encontrarse al pulsar en su nombre en la esquina superior derecha

Entrega

  • Enlace de la colección con los sketches correspondientes. Así: https://pdfcrowd.com/chatgpt-to-pdf/
  • Pdf con la conversación con ChatGPT
    • Descargarán su conversación con ChatGPT mediante la extensión https://pdfcrowd.com/chatgpt-to-pdf/
    • Solo se evalúa el código y su entendimiento sobre este, pero tendré en cuenta la conversación en las clases.
