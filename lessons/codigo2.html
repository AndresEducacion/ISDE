Operadores lógicos

bool y = false;

bool andLogico = x && y; // false (solo true si ambos son true)
bool orLogico  = x || y; // true  (true si al menos uno es true)
bool negacion  = !x;     // false (invierte el valor)

bucle while()

while (num < 10) {
  print(num);
  num++; 
}

// bucle que se repite 10 veces, mientras la variable num sea menor que diez 
se ejecutan las acciones de imprimir el número en la consola y sumarle uno.
Cuando suma 11 la condición hace que se rompa el bucle.

bucle for()

for (int num = 0; num >= 10, num++) {
  print num 
}

// Literalmente es el mismo ejemplo de la anterior diapósitiva
hecho con un bucle for() en vez de con un while(), es una manera de atajar la cuenta. 
Se usa sobre todo en acciones secuenciales.
